Prequisites: Make sure you have JDK6 or higher installed on your system and have your java and javac environment variables setup

To Compile, at the command line change into the src directory and enter the following instruction (omitting the quotes):

“javac CacheBlock.java CacheBlockDiscarder.java CacheBlockImpl.java CacheBlockSet.java CacheBlockSetImpl.java Driver.java MemoryCache.java SetAssociativeCache.java”

To run, at the command line enter the following instruction (omitting the quotes): 

“java Driver”
